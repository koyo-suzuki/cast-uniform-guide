# オリジナル制服申請ルール（イントラ掲載用ドラフト）

更新日: 2026-07-22

> この文書は公開前のドラフトです。`〔要確認〕`の項目が確定してからイントラへ掲載してください。

## 1. この制度について

条件を満たしたキャストまたはキャストマネージャーが、希望をもとにデザイナーとオリジナル制服を制作できる制度です。予算は税込10万円以内、制作は1年に1着までです。

## 2. 申請できる人・達成条件

### キャスト

- 月間売上小計150万円を3か月連続で達成すること
- 対象となる3か月に30日以上出勤すること
- リクエスト出勤は出勤日数に含めない
- ブランドアンバサダーも同じ条件を適用する

### キャストマネージャー

- キャストマネージャー就任後に申請し、在任中に月間売上小計100万円を3回達成すること
- 就任前にキャストとして達成した月は引き継がない
- 申請後、100万円を3回達成するまでの対象期間中に、店責として10日以上勤務すること
- 後から取り消しになった店責勤務は日数に含めない

### 共通

- 売上は、サービス料や手数料を引いた後の「売上小計」で判定する
- 長期休暇などで判定期間の継続が難しい場合は、申請前に運営へ相談する
- `〔要確認〕` 売上の締日、値引き・返品・イベント売上・他店舗売上の扱い
- `〔要確認〕` 出勤1日の定義、同日複数店舗勤務の数え方

## 3. 申請方法

条件の集計を始める前に、事前申請が必要です。事前申請がない場合はチャレンジを開始できません。

1. 所属店舗のキャストマネージャーから事前承認をもらう
2. チャレンジ開始日の7日前までに申請フォームを送信する
3. 運営が申請内容と判定期間を確認する
4. 承認通知を受け取ってからチャレンジを開始する

申請後の判定期間変更や、期間開始後の遡及申請は原則として受け付けません。

- `〔要確認〕` 申請の承認期限
- `〔要確認〕` 申請内容を修正・再申請できる条件
- `〔要確認〕` やむを得ない事情がある場合の例外承認者

## 4. 制作資格を失う場合

次のいずれかに該当した場合、条件達成後であっても制作資格を失うことがあります。

### 勤怠・在籍

- 判定期間中に当日欠勤が3回、または遅刻が5回以上あった場合
- 条件達成後60日以内に退店した場合、または長期休暇に入った場合
- 条件達成時点または達成後60日以内に、本人が退店意思を示し、契約協議または退店日を調整している場合
- `〔要確認〕` 遅刻と当日欠勤の定義、交通障害・療養等の例外
- `〔要確認〕` 60日の起算日と長期休暇の日数基準

### 契約・禁止行為

- 契約違反、犯罪行為、運営または店舗に損害を与える行為があった場合
- 虚偽の申請、売上・勤怠・店責勤務の改ざんや不正申告があった場合
- 店責の指示に従わず、他のキャストの売上を自分の売上として計上するなどの不正行為があった場合
- お客様への声かけやDMで、高額商品の購入を過度にあおる案内をした場合
- 他のキャストへ制服を貸した場合、または他のキャストの制服を借りた場合

### 制作進行

- 指定期限までに、デザイン希望やデザイナーからの質問へ回答しなかった場合
- 承認前にデザイナーへの依頼、発注、購入等を独自に進めた場合
- `〔要確認〕` 初回提出・修正回答の期限、催促回数、延長条件

## 5. 判定と連絡

- 売上、出勤、店責勤務、勤怠状況は、運営が指定する台帳をもとに判定します
- 達成、未達、要確認の結果を申請者へ通知します
- 不正行為や契約違反の疑いがある場合は、本人への確認後に運営が最終判断します
- `〔要確認〕` 異議申立ての期限と受付方法
- `〔要確認〕` 最終決裁者

## 6. 条件達成後の制作

- 希望するデザイン、色、モチーフ、サイズ、参考画像を提出する
- デザイナーとの打ち合わせ後、見積とデザインの承認を受ける
- 制作費は税込10万円以内とし、ヘッドドレスなどのパーツも予算に含める
- 承認前に制作を開始しない
- `〔要確認〕` 送料・採寸・初回修正費を10万円に含めるか
- `〔要確認〕` 10万円を超える場合の自己負担可否

## 7. 完成後の扱い

- 完成後2年間の制服所有者は店舗です
- 2年経過後に所有権の移転を希望する場合は、運営へ相談してください
- デザインの著作権は会社に帰属します
- 店舗での業務・公式活動以外で、制服またはデザインを無断使用することはできません
- 初回制作後のサイズ直しや通常使用による修理費は、原則として本人負担です
- `〔要確認〕` 2年の起算日、所有権移転の費用・在籍条件・承認者
- `〔要確認〕` 本人SNS、撮影会、退店後の写真・制服利用範囲

## 8. 同意事項

申請時に、次の項目への同意を必須とします。

- このルールと申請フォームに記載された条件を確認した
- 判定に必要な売上・勤怠・在籍情報を運営が確認することに同意する
- 貸し借り禁止、制作資格の失効条件、所有権、著作権、修理費用を確認した
- 虚偽申請や不正行為が確認された場合、申請取消しまたは制作資格を失うことに同意する
